# Ecuaciones Diferenciales

Material de estudio de **Ecuaciones Diferenciales (MAT1640)**.

## Contenidos

- Modelamiento con ecuaciones diferenciales
- Clasificación y métodos de solución
- Sistemas lineales de coeficientes constantes

## Indicadores de logro

1. Modelar situaciones sencillas mediante ecuaciones diferenciales.
2. Reconocer el tipo de una ecuación diferencial (lineal o no, grado, orden), identificar y utilizar los distintos métodos disponibles para la solución.
3. Calcular soluciones de ecuaciones lineales y sistemas lineales 2x2 y 3x3 de coeficientes constantes.
