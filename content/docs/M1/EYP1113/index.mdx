# Probabilidades y Estadística

Bienvenido al material de estudio de **Probabilidades y Estadística (EYP1113)**. Este módulo cubre los fundamentos probabilísticos y estadísticos necesarios para el Examen de Competencias Fundamentales.

## Contenidos

- [Probabilidad básica](/docs/M1/EYP1113/probabilidad-basica)
- [Variables aleatorias](/docs/M1/EYP1113/variables-aleatorias)
- [Estimación e inferencia](/docs/M1/EYP1113/estimacion)
- [Regresión lineal](/docs/M1/EYP1113/regresion)

## Indicadores de logro

Al finalizar este módulo, deberías ser capaz de:

1. Ajustar distribuciones de probabilidades para datos que se generen en un fenómeno de incertidumbre.
2. Describir fenómenos de incertidumbre sobre la base de variables aleatorias y hacer cálculos asociados.
3. Realizar estimaciones de parámetros de una distribución y construir intervalos de confianza.
4. Ajustar modelos de regresión lineal a fenómenos de incertidumbre.
