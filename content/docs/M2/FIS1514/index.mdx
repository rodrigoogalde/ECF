# Dinámica

Bienvenido al material de estudio de **Dinámica (FIS1514)**. Este módulo cubre los fundamentos de Estática y Dinámica necesarios para el Examen de Competencias Fundamentales.

## Contenidos

- [Estática](/docs/M2/FIS1514/estatica)
- [Movimiento lineal y angular](/docs/M2/FIS1514/movimiento)
- [Trabajo y energía](/docs/M2/FIS1514/trabajo-energia)

## Indicadores de logro

Al finalizar este módulo, deberías ser capaz de:

1. Establecer ecuaciones de movimiento y equilibrio.
2. Resolver problemas de equilibrio estático y dinámico.
3. Plantear equilibrio usando energía y trabajo virtual.
4. Manejar restricciones cinemáticas y fuerzas de vínculo.
5. Transformar fuerzas entre sistemas coordenados.
6. Conocer planteamientos numéricos para resolver mecánica clásica.
