# Cálculo I

Bienvenido al material de estudio de **Cálculo I (MAT1610)**. Este módulo cubre los fundamentos de Geometría Analítica necesarios para el Examen de Competencias Fundamentales.

## Contenidos

- [Gráficos de funciones](/docs/M1/MAT1610/graficos-funciones)
- [Derivadas](/docs/M1/MAT1610/derivadas)
- [Primitivas](/docs/M1/MAT1610/primitivas)

## Indicadores de logro

Al finalizar este módulo, deberías ser capaz de:

1. Identificar gráficos de funciones básicas, exponenciales, logarítmicas.
2. Calcular derivadas de funciones obtenidas por álgebra de funciones elementales.
3. Reconocer gráfica y analíticamente propiedades de los gráficos de funciones, como crecimiento, concavidad, máximos y mínimos locales, asíntotas.
4. Conocer el cálculo de primitivas de funciones básicas.
